# 年轮 - SIGS大事记可视化

动态的数据年轮，用户可以和年轮进行交互（比如点击不同节点出现事件，或者用手转动年轮出现事件）。
使用前端技术（HTML、CSS、JavaScript）实现。
使用Three.js实现3D效果。

## 🌟 功能特点

### 核心功能
- **3D年轮可视化** - 真实展现树木横截面的年轮结构
- **生长动画模拟** - 动态播放树木逐年生长过程
- **气候因子影响** - 模拟不同气候条件对年轮生长的影响
- **实时交互控制** - 丰富的参数调节和视觉效果控制

#### 交互体验优化
- 🎬 **启动动画** - 平滑的页面加载动画
- 📈 **进度条** - 美观的加载进度指示器
- 🎭 **悬停效果** - 丰富的鼠标悬停反馈
- 🔔 **Toast通知** - 优雅的操作反馈提示

#### 响应式设计
- 📱 **移动端适配** - 完美支持手机和平板设备
- 💻 **桌面端优化** - 大屏幕设备的最佳体验
- 🔄 **自适应布局** - 根据屏幕尺寸动态调整

## 🎮 操作指南

### 鼠标操作
- **拖拽** - 旋转3D视角
- **滚轮** - 缩放场景
- **点击年轮** - 查看详细信息

### 键盘快捷键
- `空格` - 播放/暂停生长动画
- `R` - 重置相机视角
- `H` - 显示/隐藏帮助
- `F` - 切换全屏模式
- `C` - 显示/隐藏控制面板

### 控制面板
- **基础设置** - 调节年轮数量、树干半径、年轮厚度
- **生长模拟** - 控制生长动画播放、速度和气候因子
- **视觉效果** - 切换年龄颜色、线框模式、自动旋转
- **年轮信息** - 显示点击年轮的详细数据

## 🚀 技术栈

- **JavaScript ES6+** - 现代JavaScript特性
